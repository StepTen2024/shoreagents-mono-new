# Downloads Folder

This folder contains the ShoreAgentsAI Desktop App installer.

## For Development

Place the installer here:
```
ShoreAgentsAI-Setup-1.0.0.exe
```

## For Production

The installer is hosted on GitHub Releases:
- Go to: https://github.com/YOUR-USERNAME/YOUR-REPO/releases
- Download the latest version

## Note

`.exe` files are excluded from Git commits (see `.gitignore`)
